o que é desenvolvimento web

o desenvolvimento web inclui criação, codificação e programação de sites e seus respectivos elementos e linguagens. Ele também envolve integração com sistemas e ferramentas, armazenamento de banco de dados, cibersegurança, entre outros, com o objetivo de estrurar cada detalhe de uma página da intert.

o que faz um desenvolvedor web ?

o desenvolvedor de sites, chamado de profissional de DeV ou Web Developer, é o respónsavel por criar todo o planejamento estrutural de págionas, utilizando linguagem de programação como HTML, CSS,JavaScript, PHP, entre várias outras.
